# Next.js App

## Getting Started

### Clone the Repository
```bash
git clone <repository-url>
cd <repository-name>
```

### Install Packages
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
